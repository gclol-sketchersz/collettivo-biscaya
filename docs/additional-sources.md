# Fonti Aggiuntive per Ampliamento Importazione Bandi

## Piattaforme Principali

### 1. residenzeartistiche.it
- **URL**: https://www.residenzeartistiche.it/bandi/
- **Categoria**: Residenze d'artista
- **Copertura**: Italia (nazionale e regionale)
- **Bandi Disponibili**:
  - HUMUS DEL MEDITERRANEO (Matera, 2025-2027)
  - Bottom Up 2026 (Umbria)
  - ISOLE (Etna, Sicilia)
  - Residenze Digitali 2026
  - CURA 2026 (Umbria)
  - Italia dei Visionari 2026
  - Centro Jobel 2025
  - Cirko Vertigo 2025-2027

### 2. PNRR Cultura
- **URL**: https://pnrr.cultura.gov.it/bandi-e-avvisi/
- **Categoria**: Bandi pubblici, finanziamenti PNRR
- **Copertura**: Italia (nazionale)
- **Tipo**: Grant, finanziamenti progetti culturali

### 3. Granter.it
- **URL**: https://granter.it/
- **Categoria**: Database fondazioni e bandi
- **Copertura**: Italia e internazionale
- **Tipo**: Borse di studio, grant, premi

### 4. BandoSubito
- **URL**: https://bandosubito.it/
- **Categoria**: AI-powered bandi database
- **Copertura**: Italia
- **Tipo**: Tutti i tipi di bandi

### 5. Infobandi CSVnet
- **URL**: https://infobandi.csvnet.it/
- **Categoria**: Database bandi per terzo settore
- **Copertura**: Italia
- **Tipo**: Bandi pubblici e privati

### 6. Europa Creativa
- **URL**: https://europacreativa.cultura.gov.it/
- **Categoria**: Programma europeo 2021-2027
- **Copertura**: Europa
- **Tipo**: Grant, finanziamenti progetti culturali europei

### 7. EU Funding Portal
- **URL**: https://eufundingportal.eu/it/etichetta/cultura/
- **Categoria**: Database finanziamenti UE
- **Copertura**: Europa
- **Tipo**: Tutti i tipi di finanziamenti culturali
